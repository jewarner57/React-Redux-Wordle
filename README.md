# React-Redux Wordle
- A wordle clone built using React and Redux.
- This project was built for a FEW 2.4 Redux review project assignment.

## Run Locally
- ```git clone``` the repo
- run ```npm install``` 
- run ```npm run start```

## Ideas and design based on:
- [Wordle by Josh Wardle](https://www.powerlanguage.co.uk/wordle/)
